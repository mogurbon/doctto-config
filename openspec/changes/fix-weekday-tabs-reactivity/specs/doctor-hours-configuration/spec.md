# Specification

## MODIFIED
- `resources/js/Components/DoctorSection.vue`

## DESIRED STATE
Al hacer clic en las pestañas de días (Lunes, Martes, Miércoles, etc.), el contenido asociado al día debe mostrarse instantáneamente aprovechando el sistema de reactividad de Vue 3.

## ACCEPTANCE CRITERIA
1. El clic en 'Lunes' muestra la configuración de 'Lunes' y actualiza visualmente la pestaña.
2. La interfaz reacciona instantáneamente sin recargar la vista.
3. No hay errores de consola relacionados con la reactividad en Vue.
